### Installation
#### Download docker image
```shell
docker pull bkimminich/juice-shop
```

#### Run
```shell
docker run --rm -p 3000:3000 bkimminich/juice-shop
```

#### Start
Browse to [http://localhost:3000](http://localhost:3000)
